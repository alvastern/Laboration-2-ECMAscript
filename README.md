### ECMAscript
Denna uppgift har gått ut på att jag som student har skapat en tabell med programmets alla kurser. Innehållet i tabellen har hämtats från en API med hjälp av async/await och try/catch, tabellen kan sorteras i bokstavsordning, samt filtreras baserat på sökning. Jag har använt mig av verktyget vite för automatisering, använt mig av olika branches, samt använt git och Github för versionshantering.
